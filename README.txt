En la carpeta Desarrollo software se encuentran los siguientes componentes:

- Archivos Ejecutables .exe

En la carpeta Exportaciones BD:

- Las Bases de Datos que estan conectadas a los desarrollos.

En la carpeta Exportaciones proyectos netbeans:

- Todos los proyectos desarrollados en el IDE netbeans con sus componentes.

- Para importar los desarrollos en netbeans se debe importar directamente la carpeta del "desarrollo" elegido, ejemplo: RegistrosUsuarios

Paso 1: Clic en File (se encuentra en la parte superior del IDE)
Paso 3: Clic en Import Project
Paso 4: From ZIP
Paso 5: Seleccionar la carpeta donde se encuentra el archivo a importar.

**Recordatorio: los archivos siempre deben ser importados en formato ZIP, ya que netbeans no permite abrir las carpetas contenedoras en RAR**
